Get started

Set it up in four steps

Same simple flow on both — name the computer ELD-MA, run the app, connect your phone.

1

Download & unzip

Get the app and double-click the zip → MatrackTruckSim.app.

Download for Mac · 0.9 MB · universal
2

Open it — first time only

Right-click the app → Open → Open. macOS asks once because it's an in-house app — it's safe.

Still blocked? System Settings → Privacy & Security → Open Anyway.
When prompted, allow Bluetooth.
3

Name your Mac ELD-MA

System Settings → General → About → Name → e.g. ELD-MA-You, then toggle Bluetooth off/on.

The app is found by this name — it must start with ELD-MA.
4

Connect your phone

Open the app → it reads “Advertising as ELD-MA.” On the phone (test account): ELD app → Bluetooth → pick ELD-MA → connect.

Signal & Drop

Help

Common questions

macOS won't open the app
It's an in-house app (unsigned), so macOS warns once. Right-click → Open → Open, or System Settings → Privacy & Security → Open Anyway. It's safe.
My phone doesn't see “ELD-MA”
Make sure the computer is named ELD-MA… and Bluetooth is on. Toggle the computer's Bluetooth off/on. On iPhone you can also Settings → Bluetooth → Forget the old device, then reconnect.
Only one phone connects at a time
One computer = one truck. For several people driving at the same time, each driver needs their own computer running the app (each named ELD-MA…).
The connection drops on its own
Expected if you pressed DROP (out-of-range) or ran a stored-dump at ≈500 ms. Press BACK to reconnect, or use 1 s cadence to avoid the disconnect.
